# 🛒 ASP.NET Core MVC E-Ticaret Platformu

Bu proje, ASP.NET Core MVC (.NET 9.0) kullanılarak geliştirilmiş, N-Katmanlı mimari prensiplerini ve modern web teknolojilerini barındıran tam kapsamlı bir B2C e-ticaret uygulamasıdır. Proje, gerçek dünya ihtiyaçlarını karşılayacak şekilde güvenli ödeme altyapısı, dinamik harita entegrasyonu ve rol bazlı yönetim paneli sunmaktadır.

## 🚀 Öne Çıkan Özellikler

* **Güvenli Ödeme Altyapısı:** Iyzico API entegrasyonu ile uçtan uca simüle edilmiş online ödeme sistemi.
* **Kapsamlı Yönetim (Admin Paneli):** Kategori, ürün, kullanıcı ve siparişlerin kolayca yönetilebildiği güvenli arayüz.
* **Sipariş Takip Sistemi:** Kullanıcıların sipariş durumlarını (Hazırlanıyor, Kargoda, Teslim Edildi vb.) anlık takip edebilmesi.
* **Rol Bazlı Yetkilendirme (Authorization):** Yönetici (Admin) ve standart kullanıcı (User) erişim kontrolleri.
* **Dinamik Adres Yönetimi:** Leaflet Map (Leaflet.js) entegrasyonu sayesinde harita üzerinden interaktif konum ve adres seçimi.
* **Kullanıcı Deneyimi (UX):** Güçlü Session yönetimi ile sepet işlemleri ve kullanıcı hareketlerinin optimizasyonu.
* **Veritabanı Modelleme:** Entity Framework Core (Code First) yaklaşımıyla tasarlanmış ilişkisel (RDBMS) veritabanı yapısı.
* **Responsive Tasarım:** Tüm mobil ve masaüstü cihazlarla uyumlu kullanıcı arayüzü.

## 🛠️ Kullanılan Teknolojiler & Mimari

* **Backend:** C#, ASP.NET Core MVC (.NET 9.0)
* **Veritabanı:** MS SQL Server, Entity Framework Core (Code First)
* **Frontend:** HTML5, CSS3, Bootstrap, JavaScript
* **Entegrasyonlar & Kütüphaneler:**
  * Iyzico (Ödeme Geçidi)
  * Leaflet.js (Harita ve Konum Servisi)
